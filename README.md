# typecho-material-theme

typecho material风格主题，修改自[Hanccc](https://github.com/Hanccc/typecho-material-theme)主题。

## Install
1. 把主题下载后放在`/usr/themes/`文件夹下
2. 登录后台，编辑外观，启用该主题即可

## feature

- 自定义侧边栏显示内容
- 自定义图片与标语
- 分类导航栏显示
- 备案号自定义
- Github账号自定义
- 微博链接自定义
- 添加QQ群
- 统计代码自定义
- 定义是否显示加载时间
- 手机端自适应
